# OYMO Documentation Hub

Welcome to the official documentation for the **OYMO Laser Tracker** SaaS application.

This repository contains all the necessary guides, architectural breakdowns, and reference materials needed to understand, deploy, and maintain the OYMO project.

- [Deployment Guide](./DEPLOYMENT.md) — Step-by-step instructions for deploying to Vercel/VPS, setting up Neon PostgreSQL, and configuring Backblaze B2.
